# Lea in Celeste [WIP]
 A mod to play as Lea from CrossCode in Celeste.
 
 Requires the [Everest mod loader](https://everestapi.github.io/) and [Skin Mod Helper Plus mod](https://gamebanana.com/mods/473796) to function.
